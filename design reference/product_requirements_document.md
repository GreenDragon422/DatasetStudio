**1. Executive Summary:**
This brief outlines the foundational design system for the Studio Pro Dataset Tagger application. It establishes the core visual elements, including the primary color palette and typography, to achieve a consistent, warm, retro-utilitarian, and skeuomorphic-lite aesthetic across the application. The goal is to provide clear, actionable guidelines for future UI development and ensure a cohesive user experience.

**2. Goals:**
*   To establish a consistent and branded visual identity for the Studio Pro Dataset Tagger.
*   To provide a definitive color palette and typographic hierarchy for all UI elements.
*   To cultivate a distinct "warm, retro-utilitarian, skeuomorphic-lite" aesthetic through specific color and font choices.
*   To streamline UI development by centralizing core design decisions for the Studio Pro Dataset Tagger.

**3. Target Audience:**
*   UI/UX Designers
*   Front-end Developers (specifically working with XAML/Avalonia)
*   Project Stakeholders involved in the Studio Pro Dataset Tagger.

**4. Key Design System Components:**

**4.1. Color Palette:**
The following colors form the primary palette, intended to evoke parchment-like tones with subtle borders and clear calls to action.

*   **Background Cream:** `#F9F1C7`
    *   *Intended Use:* Primary background for screens and main layouts.
*   **Muted Sand:** `#E3D5B5`
    *   *Intended Use:* Used for panels, input fields, and secondary containers.
*   **Action Orange:** `#D6611B`
    *   *Intended Use:* Primary color for buttons, interactive elements, and focus states.
*   **Deep Charcoal:** `#4A453F`
    *   *Intended Use:* Main text color for readability.
*   **Error Red:** `#D32F2F`
    *   *Intended Use:* Alerts, error messages, and critical warnings.

**4.2. Typography:**
Two distinct font families are selected to provide clear differentiation between headings and functional text, maintaining the overall aesthetic.

*   **Headings:** `Inter`
    *   *Intended Use:* For titles, section headers, and prominent display text.
*   **Code/Inputs:** `Roboto Mono`
    *   *Intended Use:* For code snippets, user input fields, and monospaced text elements.

**4.3. Visual Style / Theme:**
The overarching visual style for the Studio Pro Dataset Tagger is characterized by:
*   **Warmth:** Achieved through the predominant cream and sand tones.
*   **Retro-Utilitarian:** A functional aesthetic with a nostalgic touch.
*   **Skeuomorphic-Lite:** Incorporating subtle 3D effects, shadows, and borders reminiscent of real-world objects, but without being overly literal or heavy.
*   **Parchment-like Tones:** The base colors aim to mimic aged paper, contributing to the retro feel.
*   **Subtle Borders:** Used to define elements and add to the skeuomorphic-lite approach.
